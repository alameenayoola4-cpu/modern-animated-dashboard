# Nexus.io Admin Dashboard

A modern, animated admin dashboard built with React, Vite, and Framer Motion. Features a sleek dark theme with interactive charts, smooth animations.

## Features

- Dark theme UI with teal accents
- Smooth page transitions and micro-animations
- Interactive revenue and traffic charts
- Recent transactions table with status badges
- Top performing products with images
- Responsive layout for all screen sizes
- Login page with form validation

## Tech Stack

- React 18
- Vite
- Framer Motion
- Recharts
- React Router
- Lucide React
